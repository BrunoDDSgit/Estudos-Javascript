JavaScript - A História básica

Sua Origem

O JavaScript é um linguagem de programação que surgiu em 1995 e foi criada por Brendan Eich, anteriormente era chamada de "Livescript", mas para aproveitar a popularidade que a linguagem Java estava conseguindo, foi renomeada para JavaScript, apesar de não ter relação direta.

Utilização

O JavaScript foi desenvolvido inicialmente para ser uma linguagem de script para navegadores web, permitindo a criação de páginas web interativas e dinâmicas.
Fazendo uma analogia simples, o HTML(Hypertext Markup Language) é a estrutura da página, o CSS(Cascading Style Sheet) é o design e o JavaScript é a interatividade, ou seja, ele traz vida à página, assim complementando um ao outro. obs: das três opções, apenas o JavaScript é uma linguagem de programação.
